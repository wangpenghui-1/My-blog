## 一、你可能已经试过很多 AI 助手了

ChatGPT 你用过。它很聪明，但每次打开都是全新的——它不记得你上次说了什么，不记得你的工作习惯，不记得你偏爱简洁还是详细的回答。每次新对话都要重新介绍一遍自己，像是每天第一次见面的陌生人。

Claude、Gemini、Kimi……大同小异。它们是出色的对话工具，但本质上是"用完即走"的服务窗口，不是"陪着你的助手"。

**Hermes Agent 不一样的地方就在这里**：它运行在你自己的服务器或电脑上，24 小时在线，每次对话结束后会把有价值的信息存下来。它会记住你喜欢什么格式的总结，记住你常用的工具，记住你上周让它帮你调研的那个项目背景。你用得越久，它越懂你。

它的官方标语是"The agent that grows with you"——与你一同成长的 Agent。

这篇文章是写给想尝试但不知道从哪里开始的人的。不需要编程背景，不需要懂服务器，只需要愿意花一个下午，跟着一步一步走完。

## 二、安装前：你需要准备什么

先把清单列出来，避免中途卡住。

**① 一台电脑（Windows、Mac 或 Linux 都可以）**

Windows 用户需要额外安装一个叫 WSL2 的工具（Windows 下运行 Linux 的环境），下面会详细说。Mac 和 Linux 用户可以跳过这一步。

**② 一个 OpenRouter 账号（免费）**

OpenRouter 是一个 AI 模型中转平台，注册后可以用一个 API Key 访问几十种不同的 AI 模型，包括 Claude、GPT、Gemini 以及一些完全免费的开源模型。注册地址：`openrouter.ai`，邮箱注册，不需要付费就能开始。

**③ 一个 Telegram 账号（强烈推荐，非必须）**

Hermes Agent 配置好 Telegram 之后，你可以直接在手机上的 Telegram 里和它聊天，随时随地发任务。这是最方便的使用方式。

**④ 大约 1 小时的空闲时间**

## 三、安装：一行命令搞定（Windows 用户多一步）

### Windows 用户：先装 WSL2

1. 按 `Win + X`，选择"终端（管理员）"或"PowerShell（管理员）"
2. 输入以下命令并按回车：

```powershell
wsl --install
```

3. 等待安装完成，**重启电脑**
4. 重启后会自动弹出一个 Ubuntu 窗口，让你设置用户名和密码

装好之后，以后每次要用 Hermes 就打开"Ubuntu"应用，而不是 Windows 的 PowerShell。

### Mac / Linux 用户：直接打开终端

Mac 用户打开"终端"（Terminal）应用，Linux 用户打开你的终端即可。

### 所有人：运行安装命令

在终端/Ubuntu 窗口里，粘贴以下命令，按回车：

```bash
curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash
```

这个脚本会自动安装所有依赖，整个过程大约 3-5 分钟。看到 `✓ Hermes Agent installed successfully` 就说明安装成功了。

## 四、第一次配置：选模型、连接 Telegram

安装完成后，在终端输入 `hermes`，**第一次运行会自动弹出配置向导**。

### 第一步：选择 AI 模型提供商

选择 **OpenRouter**，然后在 `openrouter.ai` 的 API Keys 页面创建一个 Key，粘贴回终端。

### 第二步：选择具体模型

新手推荐：

| 模型 | 费用 | 特点 |
|------|------|------|
| `google/gemini-flash-1.5` | 极低（约 $0.0001/千字） | 速度快，日常任务够用 |
| `anthropic/claude-3.5-haiku` | 较低（约 $0.0008/千字） | 理解力更好，复杂任务推荐 |

完全免费体验：选带 `:free` 标记的模型，如 `google/gemma-3-27b-it:free`。

### 第三步：配置 Telegram

1. 在 Telegram 搜索 `@BotFather`，发送 `/newbot`
2. 起名后，BotFather 会给你一串 Token（`7xxxxxxxxx:AAxxxxxx...`）
3. 将 Token 粘贴进配置向导

配置完成后，在 Telegram 找到你的 bot，发送任意消息，然后在终端输入 `/sethome`，设置主频道。

完成后输入 `hermes --tui` 启动更好看的界面。**到这里，安装和配置就完成了。**

## 五、前三天：用这些任务"喂"它

### 任务 1：搜索一个感兴趣的话题

```
帮我搜索"2026年最值得关注的AI工具"，给我一个简洁的中文摘要，5个要点。
```

### 任务 2：告诉它你的偏好

```
我以后让你总结文章，都用中文，格式是：先一句话结论，然后3-5个要点，每个要点不超过两行。
```

说完之后，这个偏好会被记进用户模型，以后自动使用，无需每次重复。

### 任务 3：设置你的第一个每日简报

```
每天早上 8 点，帮我搜索一下科技行业最新新闻，整理成 5 条摘要，发到我的 Telegram。
```

它会自动创建定时任务。第二天早上 8 点，Telegram 就会收到一条来自你的 bot 的消息。

### 任务 4：布置一个真实工作任务

- "帮我整理一下这份会议纪要，列出决策事项和待办任务"
- "帮我把这段英文邮件翻译成中文，语气正式"
- "帮我写一份 XX 产品的竞品调研提纲"

## 六、第一周的难关：为什么你会想放弃

### 难关 1：它回答慢，有时候还出错

**怎么应对**：接受它不是"即问即答"工具的设定。复杂任务让它跑，你去做别的事。预算允许就换成 Claude Haiku 或 Gemini Flash。

### 难关 2：前几天感觉它跟普通 AI 没什么区别

确实如此——它的优势是随时间积累的，不是开箱即用的。

**怎么应对**：坚持每天给它几个任务，连续用满一周，你会明显感受到它开始"认识"你了。

### 难关 3：命令行界面让人不舒服

**怎么应对**：配置好 Telegram 之后，日常使用都在 Telegram 里完成，体验和聊天没有区别。命令行只需要偶尔启动一次。

### 难关 4：不知道还能让它干什么

**怎么应对**：直接问它——

```
我是一个内容创作者，每天需要收集素材、写文章、发布内容。你能帮我做哪些自动化的事情？
```

## 七、进入熟练期：5 个生产力场景

**场景一：定制每日简报**
设置多个信息源，每天早上一份定制简报推送到 Telegram。

**场景二：内容创作辅助**
它记住你的写作风格偏好后，生成的内容越来越接近你的语气，改稿越来越少。

**场景三：跨会话知识整理**
```
帮我整理一下最近三周我们讨论过的关于"AI工具选型"的内容，汇总成大纲
```
这个能力普通 AI 做不到——它们不记得上周的对话。

**场景四：周期性任务跟踪**
```
每周五下午 5 点，提醒我整理本周未完成的任务，列个清单发到 Telegram，并问我下周计划
```

**场景五：邮件与文档处理**
处理几次之后，它会学会你的邮件回复风格，越来越少需要你修改。

## 最后说一个误区

很多人以为 Hermes Agent 是"更智能的 ChatGPT"，期望打开就能用、立刻比 ChatGPT 好用。

不是这样的。

Hermes Agent 更像是一个**需要磨合的助理**，而不是一个随开随用的工具。它的价值不在于第一天，而在于第三十天。

磨合的代价是时间和耐心，收益是一个真正了解你工作方式的 AI 助手——而不是一个每次见面都把你当陌生人的对话框。

如果你愿意投这个时间，现在是一个好时机。
